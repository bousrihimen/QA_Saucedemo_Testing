1- QA Manual Testing Project – SauceDemo
Ce projet contient des tests manuels réalisés sur le site SauceDemo.

2- Objectif
Tester les principales fonctionnalités d'un site e-commerce.

3- Fonctionnalités testées 
- Connexion utilisateur (login)
- Liste des produits
- Ajout / suppression au panier
- Comportement du panier
- Processus de commande (chekout)

4- Types de tests 
- Test fonctionnel
- Test négatif
- Test UI
- Test end-to-end

5- Outils 
- Jira (suivi des bugs)
- tests manuels
